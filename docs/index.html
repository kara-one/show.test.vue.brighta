Description of the task

The project was created according to a test task

Layout the page shown in the Frame 75 design: link to layout in Figma link to PDF (menu item shown only).

!!!!!!!The work must be done exactly as on the TK layout!!! using vue.js!!!!

It is necessary to implement a filter calendar whose work shown in additional views. Explanations: the default view is "Today".

The user can choose:

  • proposed ready-made options (Yesterday, Last 7 days, Last 30 days, etc.). If the option “7 days” is selected, then the period in the last 7 days is automatically painted over on the calendar (similarly for other options). After selecting the period, the Update button is activated.

  • Select your own period immediately on the calendar (from and to), after which the Update button is activated.

After clicking Refresh, the selected period appears additionally above the table, view Frame 90.

When coding, take into account the transfer of the necessary data to the back-end. Adaptive for mobile, tablet - no need.
